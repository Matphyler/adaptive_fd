# Adaptive finite differencing

Usage:

```
Usage: python3 run.py [OPTIONS]

Options:
  -o, --order INTEGER  order of the polynomial
  --cl FLOAT           lower bound of polynomial coefficient
  --cu FLOAT           upper bound of polynomial coefficient
  --xl FLOAT           lower bound of x
  --xu FLOAT           upper bound of x
  --el INTEGER         lower bound of error (in terms of 1E<lower_bound>)
  --eu INTEGER         upper bound of error (in terms of 1E<upper_bound>)
  --out TEXT           path to output
  --help               Show this message and exit.
```
